# Программное средство для составления квизов

Оконное приложение на C++ и SFML по теме второго курсового проекта. Идея основана на Marquiz: пользователь создает маркетинговые квизы, добавляет вопросы и варианты ответов, запускает прохождение, собирает контактные заявки и формирует отчеты.

## Возможности

- регистрация и вход пользователя;
- создание квизов с категорией, темой и бонусом;
- удаление квизов вместе с вопросами, ответами и заявками;
- добавление вопросов разных форматов: варианты ответа, текстовый ответ, соответствия;
- настройка количества ответов от 1 до 10, баллов, лимита времени и изображения;
- прохождение квиза в режиме предпросмотра;
- отдельный экран выбора квиза перед прохождением;
- форма контактов после прохождения;
- список заявок, поиск, удаление заявок и формирование сертификата;
- статистика: открытия, заявки, конверсия;
- HTML-отчеты для печати;
- автоматический выбор первого квиза пользователя после входа;
- понятные сообщения при ошибках ввода и записи файлов;
- темная визуальная тема с единым оформлением экранов;
- справочная система F1.

## Хранение данных

Все данные хранятся в текстовых файлах папки `data`:

- `users.txt` - пользователи;
- `quizzes.txt` - квизы;
- `questions.txt` - вопросы, тип, путь к изображению и лимит времени;
- `answers.txt` - варианты ответов;
- `leads.txt` - заявки клиентов.

Отчеты и сертификаты создаются в папке `data/reports` как HTML-файлы. Формат основных файлов простой: поля разделяются символом `|`, поэтому приложение очищает пользовательский ввод от переносов строк и заменяет этот символ перед сохранением.

## Зависимости

- C++17;
- SFML 3;
- CMake 3.24 или новее для кроссплатформенной сборки;
- на macOS удобно установить SFML через Homebrew: `brew install sfml`.

## Запуск на macOS

```bash
make run-app
```

Обычный запуск:

```bash
make
./quizmaker_sfml
```

## Сборка на Windows

Рекомендуется Visual Studio 2022 + CMake + vcpkg. Проект использует системный шрифт Segoe UI и создает папку `data` рядом с exe-файлом.

```powershell
git clone https://github.com/microsoft/vcpkg C:\vcpkg
C:\vcpkg\bootstrap-vcpkg.bat
C:\vcpkg\vcpkg install sfml
cmake -B build -S . -DCMAKE_TOOLCHAIN_FILE=C:\vcpkg\scripts\buildsystems\vcpkg.cmake
cmake --build build --config Release
.\build\Release\QuizMaker.exe
```

## Аналог

Marquiz: конструктор квизов для сайта, сбора заявок, статистики и повышения конверсии. В проекте реализована учебная упрощенная версия этих функций.

## Надежность

Приложение проверяет числовой ввод, длину логина и пароля, повторяющиеся названия квизов, одинаковые варианты ответов и корректность телефона в заявке. Поврежденные строки в текстовых файлах пропускаются, папки данных создаются при запуске, а при ошибках записи пользователь получает понятное сообщение.
